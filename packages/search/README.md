# @nordcraft/search

[npm](https://www.npmjs.com/package/@nordcraft/search)

This contains all issue rules and search functionality for traversing a [Nordcraft](https://nordcraft.com) project. This powers the issue panel in the Nordcraft editor in a web worker atm.
For documentation about each issue rule, please visit our docs site [here](https://docs.nordcraft.com/references/issue-rules).
